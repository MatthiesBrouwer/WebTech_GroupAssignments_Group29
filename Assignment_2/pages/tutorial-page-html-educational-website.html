HTML Tutorial

HyperText Markup Language (HTML) is the standard markup language for web pages. "HyperText" refers to the hyperlinks contained in a HTML webpage, and "Markup Language" refers to HTML's use of tags to define the layout and elements of a web page.

Tag Breakdown

A tag is a generic term used for language element descriptor. HTML tags are usually defined as "a set" of characters constituting a formatted command for a web page". Most tags must have two parts. The first part is the Opening tag, consisting of an element name within angle brackets (<>), signifying the start of an element. The second part is the closing tag, consisting of the same structure as the opening tag, but with a forward-slash (/) before the element name. However, There are some tags that dont require a closing tag (i.e. images). When used in an HTML document, tags form the building blocks for HTML elements. An HTML element tells the browser what content to display and how to display it.

Attributes define additional properties of an element such as an input type, a links value reference, or the width and height of an image. Attributes are always specified within the opening tag of an element and consist of the attributes name and it's value enclosed within quotation marks (name="value").

Most common tags

There are too many elements to list, but the elements outlined below are most commonly used in web development:

  • <p>: Defines a text paragraph. Browsers automatically add a whitespace before and after a paragraph.
  • <h1>, <h2>, <h...>, <h6>: Defines a title text.
  • <a>: Defines a link. The element between the opening and closing tags can be clicked on to refer the user to another page.
  • <b>: Declares that a text should be bold.
    • <strong>: Declares a bold with emphasys.
  • <img>: Defines an image to be displayed on the page.
  • <ol>, <ul>: Define a list of elements thats either ordered or unordered.
    • <li>: Define used by lists to define a list item.

Basic HTML Document

HTML documents contain a specific syntax that tells a browser that its an HTML document, allowing it to correctly process it's content. Below shows the basic structure of a modern HTML document.

  1. The <!DOCTYPE html> declaraction describes in what version of HTML the document is written to allow a browser to interpret the text correctly. This should always be at the start of your document.

  2. The <html> and /<html> tags are the root of a web page and tells the browser that it is reading HTML code. All other contents of your page are between these tags.

  3. The <head> and /<head> tags contain information about the web page such as title, meta tags and the location of the CSS file. Most of this information is of little use to a website visitor and is not directly visible to them.

  4. The <meta> element defines metadata (information about data) about an HTML document. They are typically used to specify character sets, page descriptions, keywords, etc. Metadata is used by browsers to help disply contents, search engines for page keywords, and many other services.

  5. The <title> element declares a pages title to a browser to display at the top of the browser window.

  6. The <body> and /<body> tags contain all content that's viewable on the browser such as headings, paragraphs, hyperlinks, images tables, etc. For example, all contents within the white part of this page is within the body.
