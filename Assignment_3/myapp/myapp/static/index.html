HTML Education - Learn How to Make a Web page

Being free to surf the internet is one of the biggest benifits of living in the 21st century, but being able to create your own web content feels even better. HyperText Markup Language (HTML) gives web developers the tools necessary to freely create web pages and applications. This website will give you a basic understanding of HTML, it's history of HTML, and some of the best practices. With the help of our tutorial, even YOU can becoma a skilled web developer and create your own insightful webpages.

HTML History

This page not only contains a quick summary of the history of HTML. It will also contain an overview of the most important HTML versions and their specification. Finally there will be a segment about the current version of HTML and a compelling reveal on why it’s called “the living standard”.

Tutorial

If you want to set foot towards your first HTML page this is the place to be! You will get basic information about HTML and learn about language specific terms such as tags and elements. Some tags are more used then others, so therefore the most common tags are shown and an example is given on how to structure your first ever HTML webpage. Goodluck on your first ever HTML project!

Best practice

On this page you will find 7 easy but important tips for writing HTML for beginners. There will be tips on how to organise your code and how to optimise your page for search engines. Best Practices require you to decide consciously whether your coding is good or bad.Practise these tips while you are working on your code to become a HTML pro.
