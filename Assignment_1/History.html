The history of HTML

HTML is orginally created by Timothy Berners-Lee in late 1990. When he invented the World Wide Web in 1989, he found the need for a standardized language which could be used for formatting the displaying material retreived over the internet. That's where HTML short for Hypertext Markup Language came to existence. The first sign of HTML in public was a discriptive document called "HTML Tags" posted on the internet by the creator himself. This document contained information about the first version of HTML including 18 different elements. At first HTML was seen as an application of SGML (another formatting standard), however when time went by, HTML progressed and became a standalone concept.

HTML Timeline

Version Name Release Date Chaged Features Notes
HTML 1993 18 new different elements This was a draft version which expired in 1994
HTML 2.0 November 1994 Form-based file uploads, client size image maps, tables and more This was a working draft, which was abandoned in 2009
HTML 3.0 April 1995 Not many new technical features, most added features could be found in style formatting extentions This verion did not succeed for many reasons, including the 150 page draft
HTML 3.2 January 1997 Removal of math formulas, adopted most of Netscape's visual markup tags The first version developed and standardized exclusively by the W3C
HTML 4.0 December 1997 Adoption of many browser-specific element types Released as three variations: Strict, Transitional and Frameset
HTML 5 January 2008 Added detailed processing models to encourage more interoperable implementations The current specification is known as the HTML Living Standard and is used to create this website
